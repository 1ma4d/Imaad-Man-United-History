**#WEBSITES**

A couple of personal web projects I put together with Claude (Anthropic's AI)

I am a Maths student with no training in web design or a web dev background but wanted to see how to use Claude to make websites. 
Whilst they are relatively simple, the process was mostly prompting. For example asking what I wanted clearly and concisely, iterating on it and adjusting certain parts like titles and other design features.
